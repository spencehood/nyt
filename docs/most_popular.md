#Most Popular API

###Most Emailed

##### *Required Params*

- section
- time-period

##### *Usage*
```javascript
nyt.mostPopular.emailed(args, callback)
```

##### *Example*
```javascript
nyt.mostPopular.emailed({‘section’:’science’, ‘time-period’:’1’}, console.log)
```
